# PKFIT Carousel Template

Use this template to produce repeatable, high-retention carousels in PKFIT brand style.

## Format spec

- Aspect ratio: 4:5 (`1080x1350`) for feed
- Slide count: 8 to 10 slides
- Brand palette:
  - Background: `#080808`
  - Accent: `#C8A96E`
  - Text: `#F5F5F5`
- Font style:
  - Headline: Bebas Neue
  - Body/UI: DM Mono
- Layout rule: one core idea per slide

## Core 9-slide formula

1. Cover hook
2. Problem reframing
3. Mistake #1
4. Mistake #2
5. Mistake #3
6. System principle
7. Action framework
8. Implementation prompt
9. CTA slide
